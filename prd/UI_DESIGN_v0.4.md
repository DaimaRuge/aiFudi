# SkyOne Shuge UI/UX 设计稿 v0.4

## 1. 设计原则

### 1.1 核心设计理念
- **本地优先**: 强调数据主权与隐私
- **简洁高效**: 减少认知负担，快速完成任务
- **AI 增强**: AI 功能自然融入，不喧宾夺主
- **跨平台一致**: Web/Desktop/Mobile 体验统一

### 1.2 设计风格
- **风格**: 现代简洁，半扁平化设计
- **配色**: 
  - 主色: #1A73E8 (科技蓝)
  - 辅色: #34A853 (成功绿)
  - 警示: #EA4335 (警告红)
  - 背景: #FFFFFF / #F8F9FA
- **字体**: 系统默认字体 (中文: PingFang SC / Windows: 微软雅黑)
- **圆角**: 8px - 12px
- **阴影**: 柔和投影
