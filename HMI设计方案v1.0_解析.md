## 📖 方案概述

本方案为一套**房车智能控制系统**的 HMI（人机交互界面）设计方案，采用深色科技风格，涵盖灯光、设备、底盘控制、电器、水路、能源、影音娱乐等功能模块，以 3D 房车模型为核心可视化载体，实现直观的车载智能控制体验。

## 🎨 设计规范总结

### 色彩体系
| 角色 | 色值 | 用途 |
|---|---|---|
| 主背景 | 黑色/深灰 #000000 | 全局背景 |
| 文字/图标 | 白色 #FFFFFF | 所有可读内容 |
| 强调色 | 橙色 #FF6600 | 选中态、ON状态、高亮标注 |
| 正常状态 | 绿色 | 电量/资源充足 |
| 信息状态 | 蓝色 | 水位、中等电量 |
| 辅助色 | 紫色 | 氛围灯、渐变装饰 |
| 关闭状态 | 灰色 | OFF状态、未选中标签 |

### 交互规范
- **开关（Toggle）**：圆形滑块，ON=橙色右置，OFF=灰色左置
- **标签页**：深灰背景，选中项底部橙色下划线
- **滑块（Slider）**：黑色轨道+白色滑块，左右 +/- 按钮
- **方向控制**：圆盘式设计，四方向箭头

### 导航体系
- **底部导航栏**（7项，贯穿所有页面）：灯光 | 设备 | 水路 | 主页 | 空调 | 能源 | 影音
- **页面内标签**：每个功能模块内通过标签页切换子分类
- **返回导航**：左上角白色箭头

### 布局模式
- **标准布局**：顶部状态栏 + 左侧控制面板 + 右侧3D房车俯视图 + 底部导航栏
- **3D模型联动**：控制操作与右侧车辆模型高亮联动，直观展示控制对象位置
