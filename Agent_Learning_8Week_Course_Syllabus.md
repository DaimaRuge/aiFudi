# Agent Learning 8周系统学习课程大纲

## 课程概述

**课程名称**: Agent Learning 系统开发课程  
**课程时长**: 8周 (56天)  
**学习目标**: 掌握 Agent 系统开发全栈技能，从基础工具执行到多 Agent 协作  
**适用对象**: 有一定编程基础的开发者、AI工程师、系统架构师  
**难度等级**: 中级到高级  
**学习平台**: Claude Code (learn.shareai.run)  
**课程版本**: v2.0 (2026-03-23)

## 课程结构

### 阶段一：基础能力建设 (第1-2周)

#### 第1周：Agent 核心架构
**目标**: 掌握 Agent 基础循环和工具执行机制

**学习内容**:
- **L1-s01**: The Agent Loop - Bash is All You Need (84 LOC)
  - 最小 Agent 内核实现
  - while 循环 + 工具调度基础
  - 理解 Agent 工作流程
  
- **L1-s02**: Tools - One Handler Per Tool (120 LOC)
  - 工具注册与调度映射
  - 统一工具接口设计
  - 错误处理与异常管理

**实践项目**: 文件操作 Agent
- 实现文件读写工具
- 实现目录管理工具
- 构建完整命令行 Agent

#### 第2周：任务规划系统
**目标**: 学习任务分解与规划，避免 Agent 漂移

**学习内容**:
- **L2-s03**: Todo - Write Plan Before You Act (176 LOC)
  - 任务规划的重要性
  - 待办列表设计与实现
  - 先规划后执行的最佳实践

- **L2-s04**: Subagents - Clean Context Per Subtask (151 LOC)
  - 子 Agent 上下文隔离
  - 消息传递机制
  - 主对话清洁保持

- **L2-s05**: Skills - Load on Demand (187 LOC)
  - 按需技能加载
  - 知识注入策略
  - 系统提示优化

**实践项目**: 旅行规划 Agent
- 任务分解与依赖分析
- 子任务分配与执行
- 进度跟踪与报告

### 阶段二：系统优化 (第3-4周)

#### 第3周：内存管理
**目标**: 在上下文限制内实现无限会话

**学习内容**:
- **L3-s06**: Compact - Three-Layer Compression (205 LOC)
  - 三层压缩策略原理
  - 短期/中期/长期记忆管理
  - 摘要生成与重要性评分

**实践项目**: 智能客服记忆系统
- 实现会话摘要生成
- 设计重要性评分算法
- 测试记忆保留效果

#### 第4周：并发处理
**目标**: 掌握非阻塞执行和并行工作

**学习内容**:
- **L4-s07**: Tasks - Task Graph + Dependencies (207 LOC)
  - 任务图数据结构
  - 依赖关系管理
  - 并行任务识别

- **L4-s08**: Background Tasks - Background Threads + Notifications (198 LOC)
  - 后台线程执行
  - 事件通知机制
  - 非阻塞操作实现

**实践项目**: 数据爬取系统
- 设计并行爬取任务
- 实现任务依赖管理
- 构建进度监控界面

### 阶段三：高级应用 (第5-7周)

#### 第5周：团队协作基础
**目标**: 实现多 Agent 通信与协作

**学习内容**:
- **L5-s09**: Agent Teams - Teammates + Mailboxes (348 LOC)
  - 团队组建与管理
  - 邮箱通信系统
  - 任务委托机制

- **L5-s10**: Team Protocols - Shared Communication Rules (419 LOC)
  - 团队通信协议设计
  - 请求-响应模式
  - 冲突解决策略

**实践项目**: 客服团队协作系统
- 构建多 Agent 客服团队
- 实现任务分配与负载均衡
- 设计团队协商协议

#### 第6周：自主 Agent 系统
**目标**: 实现自主任务认领与执行

**学习内容**:
- **L5-s11**: Autonomous Agents - Scan Board, Claim Tasks (499 LOC)
  - 任务板设计与实现
  - 自主任务认领机制
  - 状态监控与报告

**实践项目**: 项目管理 Agent 系统
- 设计任务板界面
- 实现自主任务分配
- 构建进度监控面板

#### 第7周：工作隔离与安全
**目标**: 掌握工作树隔离和系统安全

**学习内容**:
- **L5-s12**: Worktree + Task Isolation - Isolate by Directory (694 LOC)
  - 工作树隔离机制
  - 目录级安全隔离
  - 资源访问控制

**实践项目**: 多租户 Agent 平台
- 实现用户工作空间隔离
- 设计资源配额管理
- 构建安全审计系统

### 阶段四：综合实践 (第8周)

#### 第8周：毕业项目
**目标**: 综合运用所学知识构建完整系统

**项目要求**:
1. **系统设计**: 设计一个完整的多 Agent 协作系统
2. **功能实现**: 包含至少 5 个核心功能模块
3. **性能优化**: 实现内存压缩和并发处理
4. **测试验证**: 编写完整的测试用例
5. **文档编写**: 提供详细的设计文档和使用说明

**可选项目方向**:
- 智能客服系统
- 项目管理平台
- 数据分析管道
- 自动化测试框架
