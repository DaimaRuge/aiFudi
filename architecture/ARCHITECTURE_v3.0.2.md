## 系统概述

SkyOne Shuge 是一个 AI 驱动的个人知识管理系统，帮助用户组织、连接和发现知识。

---

## 技术栈

### 后端
- Python 3.10+
- FastAPI (Web 框架)
- SQLAlchemy (ORM)
- Pydantic (数据验证)
- Alembic (数据库迁移)

### 前端
- Streamlit (UI 框架)

### AI/ML
- OpenAI API (GPT 模型)
- LangChain (AI 应用框架)
- ChromaDB (向量数据库)

### 数据库
- SQLite (开发环境)
- PostgreSQL (生产环境)

## 核心模块设计

### 1. Agents 模块
- **DocumentProcessorAgent**: 处理 Markdown 文档，提取标题、关键词，创建文档块
- **BaseAgent**: Agent 基类框架

### 2. Core 模块
- **Auth**: 密码哈希与验证
- **Config**: 环境变量配置管理
- **Database**: SQLAlchemy 引擎与会话管理
- **Dependencies**: FastAPI 依赖注入
- **Exceptions**: 自定义异常

### 3. ML 模块
- **Embedding**: 文本向量化基类与 Dummy 实现
- **LLM**: 大语言模型基类与 Dummy 实现
- **VectorDB**: 向量数据库基类与 Dummy 实现

### 4. Models 模块
- **User**: 用户模型
- **Document**: 文档模型
- **Tag**: 标签模型
- **DocumentTag**: 文档-标签关联表
