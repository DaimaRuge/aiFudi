### 2. 核心服务详细设计

#### A. 文档服务 (Document Service)

**职责**：
- 文档上传、解析、索引
- 多格式文档支持
- OCR处理
- 文档元数据提取

**技术栈**：
- **语言**: Python 3.11+
- **框架**: FastAPI
- **文档解析**: Apache Tika, PyMuPDF, python-docx
- **OCR**: Tesseract 5 + EasyOCR
- **任务队列**: Celery + Redis
- **存储**: MinIO (文档原始文件), PostgreSQL (元数据)

**API设计**：
```python
# 文档上传和解析
POST /api/v1/documents/upload
POST /api/v1/documents/parse
GET  /api/v1/documents/{doc_id}/content
GET  /api/v1/documents/{doc_id}/metadata

# OCR处理
POST /api/v1/documents/{doc_id}/ocr
GET  /api/v1/documents/{doc_id}/ocr-status

# 批量操作
POST /api/v1/documents/batch-upload
POST /api/v1/documents/batch-parse
```

**数据模型**：
```sql
-- 文档表
CREATE TABLE documents (
    id UUID PRIMARY KEY,
    user_id UUID NOT NULL,
    title VARCHAR(500),
    original_filename VARCHAR(500),
    file_size BIGINT,
    mime_type VARCHAR(100),
    storage_path VARCHAR(1000),
    parsed_content TEXT,
    ocr_content TEXT,
    language VARCHAR(10),
    page_count INTEGER,
    word_count INTEGER,
    metadata JSONB,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 文档分块表 (用于向量化)
CREATE TABLE document_chunks (
    id UUID PRIMARY KEY,
    document_id UUID REFERENCES documents(id),
    chunk_index INTEGER,
    chunk_text TEXT,
    chunk_embedding VECTOR(1536),  -- OpenAI embedding维度
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 3. 数据流设计

#### 文档处理流水线
```
1. 上传 → 2. 格式检测 → 3. 解析 → 4. 分块 → 5. 向量化 → 6. 索引

      ↓
元数据提取
      ↓
知识图谱更新
```

```
1. 用户查询 → 2. 意图识别 → 3. 知识检索 → 4. 上下文构建
      ↓
5. AI生成响应 → 6. 记忆存储 → 7. 学习路径更新 → 8. 反馈收集
```

#### 知识扩展流程
```
1. 新文档加入 → 2. 实体提取 → 3. 关系发现 → 4. 图数据库更新
      ↓
5. 外部知识搜索 → 6. 相关性评估 → 7. 自动下载 → 8. 集成到系统
```

### 5. 安全架构设计

#### 数据加密
```python
# 端到端加密
from cryptography.fernet import Fernet

class DocumentEncryption:
    def __init__(self, user_key):
        self.cipher = Fernet(user_key)
    
    def encrypt_document(self, content):
        return self.cipher.encrypt(content.encode())
    
    def decrypt_document(self, encrypted_content):
        return self.cipher.decrypt(encrypted_content).decode()
```
