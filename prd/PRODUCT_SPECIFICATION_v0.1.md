# SkyOne Shuge (天一阁) PRD v0.1
**版本日期**: 2026-02-03
**状态**: 初始草案

## 1. 产品概述

### 1.1 产品愿景
天一阁是一个基于大模型的智能个人数字文献管理与认知增强平台，帮助用户建立格式化、有序的个人知识索引体系，并通过 AI 深化认知结构。

### 1.2 核心定位
- **数字图书馆**: 格式化、有序的文献索引
- **认知分析器**: 通过 AI 深化个人知识结构
- **创作工场**: 基于知识体系生成各类内容

### 1.3 目标用户
- 学术研究者
- 知识工作者
- 专业人员（律师、医生、会计）
- 学生与教育工作者

## 2. 核心功能

### 2.1 LibIndex 数字文献索引系统

#### 2.1.1 设备级索引 (LibIndex Device)
- **文献扫描**: 扫描本地存储或云端网盘的数字文献
- **格式支持**:
  - 文档: PDF, DOCX, PPTX, XLSX, TXT, MD, JSON, XML, HTML, EPUB
  - 音频: MP3, WAV, FLAC, M4A, AAC
  - 视频: MP4, AVI, MKV, MOV, WEBM
  - 代码: Python, JavaScript, Java, C/C++, Go, Rust 等
- **智能命名**: 根据用户自定义或专业图书命名系统进行编号、命名
- **动态监测**: 实时监测文献变化，更新索引

#### 2.1.2 云端同步 (LibIndex One)
- **跨设备同步**: 多设备索引合并
- **同步策略**: 支持同步到指定设备或实时同步所有设备
- **隐私设计**: 服务器端仅提供同步功能，不存储原始索引数据
- **手动合并**: 支持将不同设备的 LibIndex 导入本地进行合并

### 2.2 AI 智能分类与元数据增强

#### 2.2.1 自动分类
- 基于大模型的智能分类引擎
- 支持对话式或界面操作进行重新分类
- 分类依据:
  - 当代学科体系（数学、物理、计算机科学等）
  - 学术惯例（期刊分类、研究主题）
  - 产业界标准（行业分类、技术领域）
  - 个人习惯与偏好

#### 2.2.2 元数据填充
- **自动检索**: 搜索公共图书馆、学术出版机构、知识问答平台等
- **智能推荐**: 基于已藏文献推荐相关版本、关联文献
- **字段完善**: 自动补充作者、ISBN、出版日期、摘要、关键词等

### 2.3 无涯子 (Wuya Study) - 认知结构深化模块

#### 2.3.1 核心理念
- 个人文献索引即个人认知结构的映射
- 通过 AI 分析收藏文献，识别知识体系结构
- 提供认知结构深化建议

#### 2.3.2 NotebookLM 式知识处理
- **多模态输入**: 语音对话、文本输入
- **知识生成**: 问答、知识总结、概念图谱
- **内容创作**:
  - 视频、音频、报告、PPT
  - 思维导图、信息图、数据表
  - 程序代码、测试题目、考试试卷

### 2.4 教育体系模块

#### 2.4.1 学制配置
- 支持各地区学制自定义
- 覆盖: 学前教育 → K12 → 高等教育 → 研究生

#### 2.4.2 专业学习模块
- 硬件设计、软件开发、产品设计
- 法律执业、会计执业、医学执业

### 2.5 项目级文献管理
- 为每个项目建立独立的文献索引空间
- 项目文献与个人藏书体系的关联与隔离
- 项目进度跟踪与文献关联

## 3. 技术架构

### 3.1 大模型底座 (可配置)
- OpenAI GPT-4
- Claude 3
- Google Gemini
- 国内大模型（文心一言、通义千问、智谱 GLM 等）

### 3.2 数据存储
- **本地优先**: 文献和索引数据主要存储在用户设备
- **云端同步**: LibIndex One 作为可选的同步服务
- **隐私保护**: 云端不存储原始索引数据

### 3.3 系统兼容性
- Windows (x86_64, ARM64)
- macOS (x86_64, Apple Silicon)
- Linux (x86_64, ARM64, RISC-V)
- Android / iOS 移动端
