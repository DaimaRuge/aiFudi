## 🏗️ 技术架构核心要点

### 整体架构
```
客户端层 → API网关层 → 业务服务层 → 数据层
```

### 核心服务（7个微服务）
1. **文档服务**：文档上传、解析、OCR、元数据提取
2. **知识服务**：知识图谱、实体识别、关系提取、图数据库
3. **学习服务**：学习路径规划、内容生成、进度跟踪
4. **AI伴侣服务**：个性化对话、长期记忆、进化学习
5. **内容服务**：多模态内容生成、课程内容管理
6. **用户服务**：用户管理、OAuth2认证、RBAC权限
7. **支付服务**：订阅管理、支付处理、发票系统

### 数据存储
- **向量数据库**：ChromaDB（文档向量化检索）
- **图数据库**：Neo4j（知识图谱）
- **对象存储**：MinIO（原始文件）
- **关系数据库**：PostgreSQL（元数据）
- **搜索**：Elasticsearch（全文搜索）
- **缓存**：Redis（热点数据）
- **时序数据库**：TimescaleDB（时间序列数据）
- **消息队列**：RabbitMQ（异步处理）
