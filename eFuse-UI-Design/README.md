# eFuse Mini 智能插座 UI 设计方案

## 设计理念

**深色科技风** — 体现电力/能源产品属性，深蓝黑底色搭配电力蓝品牌色。

## 设计规范

| 维度 | 值 |
|------|-----|
| 主背景色 | #0a0e27（深蓝黑） |
| 关闭状态背景 | #1a1a2e（深灰蓝） |
| 品牌色/强调色 | #00D2FF（电力蓝） |
| 主文字色 | #FFFFFF |
| 次文字色 | rgba(255,255,255,0.8) |
| 禁用文字色 | rgba(255,255,255,0.6) |
| 卡片背景 | rgba(255,255,255,0.05) |

## 页面说明

### 1. 开启状态 (efuse-mini-ui-on.png)
- 深蓝黑背景 + 电路纹理 + 能量流动光线
- 中央发光能量圆环动画
- 白色文字 + 微发光效果

### 2. 关闭状态 (efuse-mini-ui-off.png)
- 深灰背景 + 静态电路板纹理
- 中央暗灰色圆环（无发光）
- 柔和灰白文字

### 3. 用电量监控 (efuse-mini-ui-power.png)
- 深蓝黑背景 + 网格图案
- 中央蓝色大数字显示电量
- 底部三栏半透明卡片展示实时参数

## 设计原则
- 开/关状态背景**有明显视觉差异**，0.5秒内感知设备状态
- 深色背景下保证文字高对比度
- 品牌色统一贯穿所有页面
